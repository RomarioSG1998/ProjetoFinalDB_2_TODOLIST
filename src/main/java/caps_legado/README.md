# Legacy Package (caps_legado)

Este diretório contém a estrutura original e "legada" do projeto.

### Nota:
Este pacote foi mantido a pedido do usuário para coexistir com a nova estrutura padronizada em `com.capstone`. 
Embora funcional, recomenda-se o uso da nova estrutura para futuros desenvolvimentos, pois ela segue melhores práticas como a separação por camadas (Service Layer).

# Service Layer

Este diretório contém os **Services** da aplicação (Lógica de Negócio).

### Responsabilidades:
- Implementar as regras de negócio da aplicação.
- Intermediar a comunicação entre os Controllers e os Repositories.
- Garantir que as operações sejam atômicas e seguras.
- Isolar a complexidade da aplicação da camada de exposição (Controller).
